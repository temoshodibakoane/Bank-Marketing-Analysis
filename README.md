 Bank Marketing Campaign Analysis

 Project Overview
This project analyzes the **Bank Marketing dataset** to uncover key factors that influence whether clients subscribe to a term deposit. Using SQL, Excel, and Power BI, the data was cleaned, transformed, and visualized to help improve marketing strategies and target potential customers more effectively.

Objective
To identify patterns in customer demographics and campaign performance that drive successful marketing outcomes for term deposit subscriptions.

 Key Insights
- Age group **30–50** showed the highest subscription rates.
- Customers contacted via **cellular channels** were more likely to respond positively.
- Longer call durations were directly associated with a higher chance of conversion.
- Previous campaign outcomes had a strong impact on current success rates.

Tools & Technologies
- SQL – Data extraction, cleaning, and transformation  
- Excel – Initial exploration and preprocessing  
- Power BI – Interactive dashboard and data visualization  

Skills Applied
SQL, Power BI, Data Cleaning, Data Visualization, ETL, Business Intelligence, Dashboard Design

Dashboard Highlights
The Power BI dashboard includes:
- Customer Demographics Overview
- Campaign Success by Contact Type
- Subscription Rate by Job & Marital Status
- Monthly Trend of Term Deposit Subscriptions
